# Normaliser.java

## Description

The `Normaliser` class in Java is used to normalize profession names.

## Features

- **Data Normalization**: The class provides a method that takes a profession name in string format and returns a normalized profession name.

## How to Use

1. **Import the Class**: Ensure that the `Normaliser` class is in your Java project.
2. **Instantiate the Class**: Create an instance of the `Normaliser` class.
3. **Call the Normalization Methods**: Use the provided methods to normalize your data.

### How to Run Unit Tests

1. **Have the Tools to Run Tests**: Ensure that Java and Maven are installed and configured.
2. **Navigate to the Project Root Folder**: Go to the folder where the `pom.xml` file is located.
3. **Run the Tests**: Execute the command in the terminal `mvn test`.

## Requirements

- Java 8 or higher
- Maven 3.6 or higher
